### manualy install

1. git clone

```
git clone git@github.com:wilf312/approval-counter.git

or

git clone https://github.com/wilf312/approval-counter.git
```

2. chromeで拡張機能画面を開く

メニューバー → ウィンドウ → 拡張機能

3. 拡張を追加

パッケージ化されていない拡張機能を追加 ボタンを押す → clone したリポジトリ フォルダを選択
